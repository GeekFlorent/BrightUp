# ⚡ BrightUp | Global Social Exchange

**BrightUp** is a high-performance, mobile-first social platform designed with the **2026 Telegram "Liquid Glass" aesthetic**. It bridges global communication with a sleek, transparent UI and a robust Supabase backend.

## ✨ Core Features
* **Liquid Glass UI:** Modern refraction effects, transparent headers, and blurred navigation panels.
* **Universal Auth:** Flexible login system supporting Email, Username, or Phone Number.
* **Global Explore Feed:** Real-time broadcasting and feed updates via Supabase.
* **Privacy First:** Integrated "Spoiler" text features and encrypted-style messaging UI.
* **Full Profile Mapping:** Captures and displays Phone, Location, Gender, and Birthdate.

## 🛠️ Tech Stack
* **Frontend:** HTML5, Tailwind CSS (Styling), Lucide (Icons).
* **Backend/Database:** Supabase (PostgreSQL + Auth).
* **Design Language:** Plus Jakarta Sans & Telegram 2026 Design System.

## 🚀 Deployment
This project is optimized for **GitHub Pages**. 
1. Connect your Supabase `URL` and `ANON_KEY` in `index.html`.
2. Enable GitHub Pages in the repository settings.
3. Your app is live!
